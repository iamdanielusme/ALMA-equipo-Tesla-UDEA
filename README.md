# ALMA-equipo-Tesla-UDEA
Este repositorio contiene el desarrollo del proyecto ALMA, realizado por el equipo Tesla, en el marco del curso de Diseño de Productos Electrónicos.

ALMA es un altavoz inteligente de escritorio orientado a la interacción por voz, la reproducción de contenido multimedia y el control de dispositivos en red local. El proyecto abarca el diseño conceptual y técnico del sistema, incluyendo la definición de requisitos, arquitectura, análisis de subsistemas y futuras etapas de implementación.

En este repositorio se documentan las diferentes fases del desarrollo, comenzando por la especificación de requisitos funcionales, no funcionales, de interfaz gráfica, ambientales y normativos, junto con sus respectivos criterios de verificación.

El objetivo del proyecto es desarrollar una solución integral que combine hardware y software, aplicando principios de ingeniería electrónica para la construcción de un sistema interactivo, eficiente y orientado a uso en entornos domésticos y de oficina.
